
# Description

[Crave-tracker](https://github.com/lordgape/crave-tracker) Graphql Application that tracks milestone.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev
```

## Test

```bash
# unit tests
$ npm run test
```

```bash
# test coverage
$ npm run test:cov
```
